The Danish address picker, upgraded.

A drop-in <adressevaelger-search> web component (and headless library) for adressevaelger.dk, DAWA's successor — with the relevance ranking, unit expansion and typo tolerance the raw API leaves out.

Try it

Type a real Danish address — e.g. askevænget 9 2 tv, brogaardsvej 102 (note the aa), or a street + city like askevænget vejle.

Quick start

Web component (npm)

npm install adressevaelger-enhanced lit
import 'adressevaelger-enhanced'

// <adressevaelger-search token="your-token"></adressevaelger-search>
document
  .querySelector('adressevaelger-search')
  .addEventListener('adresse-selected', (e) => {
    console.log(e.detail.suggestion, e.detail.address)
  })

Headless library

import { createAddressSearch } from 'adressevaelger-enhanced/lib'

const search = createAddressSearch({ token: 'your-token' })
const controller = new AbortController()
const results = await search.search(controller.signal, 'askevænget 9 2 tv')
const full = await search.getAddressById(results[0].id)

Token & terms: the service needs a token query parameter. Any string of ≥10 characters works today, but that may change — request a real token from support@kds.dk and honour the service's terms. No token ships with this library. This demo uses a throwaway placeholder.
